Metric/Imperial Converter

Convert between common metric and imperial units with ease!

How It Works

This converter allows you to input values in either metric or imperial units, and it will return the converted value along with the correct unit. Simply enter a value and its unit (e.g., 3.1mi or 4gal), and the system will do the rest!

Examples of valid input:

  • [project url]/api/convert?input=4gal
  • [project url]/api/convert?input=1/2km
  • [project url]/convert?input=5.4/3lbs
  • [project url]/api/convert?input=kg

Example Output:

Example return for /api/convert?input=3.1mi:

{ "initNum": 3.1, "initUnit": "mi", "returnNum": 4.98895, "returnUnit": "km", "string": "3.1 miles converts to 4.98895 kilometers" }
